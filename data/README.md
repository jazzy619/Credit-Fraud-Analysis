# Credit Card Fraud Analysis

## The Problem
Global credit card fraud costs the financial industry over $33 billion 
annually. This project analyses 1.29 million real world transactions to 
identify which merchant categories, times of day and customer profiles 
are most associated with fraudulent activity.

## Tools Used
- SQL (Microsoft SQL Server) — data exploration and fraud pattern analysis
- Excel — data cleaning, pivot analysis and dashboard (in progress)
- Tableau — interactive dashboard (coming soon)

## Key SQL Findings
- 0.579% of transactions are fraudulent but the average fraud transaction 
  is 8x higher in value than a legitimate one
- Online shopping and miscellaneous categories have the highest fraud rates
- Grocery in store transactions are a surprising exception — likely driven 
  by card skimming at physical terminals
- Fraud peaks at 10pm and 11pm — not midnight as initially expected
- Two distinct fraud profiles identified — ghost cards and compromised 
  active cards
- Grocery in store transactions at 10pm show a 44% fraud rate — 76 times 
  above the overall baseline
